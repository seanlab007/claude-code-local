# Claude Code.command 无法打开？

## 方法一：右键打开（最简单，30秒解决）

1. **右键点击** `Claude Code.command` 文件（不要双击）
2. 选择 **「打开」**
3. 弹出对话框后，点击 **「打开」** 按钮
4. 以后就可以直接双击了

---

## 方法二：终端命令解除限制（一次性）

打开 Mac 终端（Terminal），粘贴以下命令并按回车：

```bash
xattr -d com.apple.quarantine ~/Desktop/Claude\ Code.command && chmod +x ~/Desktop/Claude\ Code.command
```

然后就可以双击打开了。

---

## 方法三：系统设置（适合不熟悉终端的用户）

1. 打开 **系统设置**（System Settings）
2. 点击 **隐私与安全性**（Privacy & Security）
3. 向下滚动，找到 "已阻止使用 Claude Code.command" 的提示
4. 点击 **「仍要打开」**（Open Anyway）
